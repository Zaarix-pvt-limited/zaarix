PROJECT BUILD REPORT

OVERVIEW

The project is a multi-service AI-powered multimedia application
composed of:

1.  Authentication Service (Independent Backend)
2.  Video Generation System (Frontend + Backend)
3.  AI-based Creative Tools (Chat-to-Video, Chat-to-Meme)

The system enables authenticated users to generate AI-driven content
including scripts, voiceovers, memes, and rendered videos.

------------------------------------------------------------------------

1. AUTHENTICATION SERVICE

Tech Stack: - Node.js - Express.js - MongoDB - JWT (Access & Refresh
Tokens) - Bcrypt - Nodemailer

Status: Functional & Secured Backend Service

Key Features:

User Authentication: - User Registration with Email + OTP Verification -
Login with BCRYPT Password Hashing - JWT-based Session Management
(Access & Refresh Tokens) - Secure Logout Mechanism

Security Measures: - Rate Limiting (express-rate-limit) - CORS
Configuration - Helmet Security Headers - Email-based Password Reset
Flow

3. AUTHENTICATION FRONTEND (Agentweb/src)

Tech Stack: React, Tailwind CSS, Axios, Lucide React

Key Components:
-   AuthPage.tsx: Integrated Login & Registration interface with toggle
    functionality.
-   ProtectedRoute.tsx: Higher-order component protecting dashboard routes
    from unauthorized access.
-   useAuthBootstrap.ts: Custom hook for session persistence and token
    verification on app load.
-   Axios Interceptors: Automatic token attachment and error handling for
    API requests.

Architecture: - MVC Pattern (Models, Controllers, Services, Routes)

------------------------------------------------------------------------

2. VIDEO GENERATION SYSTEM (/videoModel)

A. VIDEO BACKEND (/videoModel/Backend)

Tech Stack: - Node.js - Express.js - MongoDB - Google Gemini AI -
ElevenLabs API

Status: Fully Functional Backend Service

AI Integrations:

1.  Google Gemini:

-   Chat script generation
-   Conversational story building
-   Structured dialogue formatting

2.  ElevenLabs:

-   Text-to-Speech Conversion
-   Voice Generation
-   Audio File Output

API Endpoints: - POST /api/chat → Generates AI script content - POST
/api/audio → Generates voice/audio files - Static asset serving for
audio/media files

------------------------------------------------------------------------

B. VIDEO FRONTEND (/videoModel/frontend)

Tech Stack: - React (Vite) - Tailwind CSS - Remotion (Video Rendering
Engine)

Status: Frontend UI Application Complete

CREATIVE FEATURES

1.  Chat-to-Video System:
-   User inputs chat messages or script
-   AI generates structured dialogue
-   ElevenLabs generates voice
-   Remotion renders final downloadable video

2.  Chat-to-Meme Page:

-   User enters chat or dialogue
-   AI extracts punchline
-   Meme template applied
-   Meme image generated & downloadable

DASHBOARD SYSTEM

Overview:
A comprehensive dashboard accessible via a responsive Sidebar navigation,
featuring nested routing and a modern UI.

Implemented Pages:

1.  Service Audio Page (/service/audio):
    -   UI: Replicates ElevenLabs Voice Library.
    -   Features:
        -   Search Bar & Category Filters (Language, Accent, etc.).
        -   Trending Voices section with gradient avatars and playback tags.
        -   Handpicked Collections (e.g., "Popular Tiktok voices").
        -   Weekly Spotlight for character voices.

2.  Service Video Page (/service/video):
    -   UI: Video creation hub and community gallery.
    -   Features:
        -   "Create New Video" CTA with decorative banner.
        -   Community Video Gallery displaying thumbnails, creator avatars,
            views, and duration.
        -   Hover effects with playback overlay.

3.  Service Avatar Page (/service/avatar):
    -   UI: Avatar management library.
    -   Features:
        -   Grid view of AI avatars featuring status indicators (Active/Processing).
        -   Hover actions for Download and Add.
        -   Search and Filter functionality.

4.  Billing Page (/billing):
    -   UI: Modern Pricing Table.
    -   Features:
        -   Three Tiers: Basic ($0), Pro ($29), Enterprise ($99).
        -   Feature comparison lists with checkmarks.
        -   Visual highlighting for the "Pro" plan.

5.  Profile Page (/profile):
    -   User profile management interface linked from the sidebar footer.

6.  Settings Page (/setting):
    -   Configuration for application preferences.

7.  Dashboard Home:
    -   Main landing area for authenticated users.

------------------------------------------------------------------------

EXECUTION FLOW

1.  User Authentication: Handled independently by the Auth Backend.

2.  Content Creation: Users log in and access the videoModel frontend
    dashboard.

3.  AI Processing: Frontend → Video Backend →

    -   Gemini generates scripts
    -   ElevenLabs generates audio

4.  Media Generation:

    -   Audio files saved
    -   Remotion renders video
    -   Assets stored & displayed in dashboard

5.  Download & Management:

    -   Download video
    -   Download audio
    -   Manage avatars
    -   Track billing
    -   View usage history
